# Ping-Pong 2
New version of my FPGA Ping Ping game project. This version is developed for educational purposes to master SystemVerilog.

### Technical details

VGA module is implemented using two Finite State Machines (FSMs). Two separate FSM are used for each scanning direction. While using FSMs for VGA timing is not common approach, it was chosen as an exercise to understand application of FSMs in digital logic.
